# Domain

## Что делает библиотека

`email-sender` — Java библиотека для отправки HTML писем с чеком покупки.


## Основные сущности

**ReceiptEmailDTO** — данные чека, который отправляется покупателю:
- email покупателя
- номер чека
- дата покупки
- список товаров
- итоговая сумма

**ItemDTO** — одна позиция в чеке:
- название товара
- количество
- цена за единицу
- сумма по позиции

**EmailConfig** — настройки SMTP подключения:
- хост, порт
- логин и пароль приложения

**EmailService** — основной класс библиотеки, отправляет письмо.

**ReceiptTemplate** — генерирует HTML письмо с таблицей товаров.

## Ограничения

В текущей версии:
- поддерживается только Gmail SMTP
- нет поддержки вложений
- нет очереди отправки
- нет повторных попыток при ошибке
