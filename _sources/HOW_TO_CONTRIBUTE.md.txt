# How to Contribute

## Репозиторий

GitHub: [devstarash/email-sender](https://github.com/devstarash/email-sender)

## Ветки

- `master` — стабильная версия
- `feature/*` — новая функциональность
- `fix/*` — исправление багов

## Как внести изменения

1. Сделай форк репозитория
2. Создай ветку: `git checkout -b feature/my-feature`
3. Внеси изменения
4. Закоммить: `git commit -m "feat: описание"`
5. Запушь: `git push origin feature/my-feature`
6. Открой Pull Request в `master`

## Стиль коммитов

| Префикс | Когда использовать |
|---|---|
| `feat:` | новая функциональность |
| `fix:` | исправление бага |
| `chore:` | служебные изменения |
| `docs:` | документация |
| `refactor:` | рефакторинг |
